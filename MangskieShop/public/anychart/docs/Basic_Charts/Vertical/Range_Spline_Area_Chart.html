Vertical Range Spline Area Chart

  • Overview
  • Quick Start

Overview

This article explains how to create a Vertical Range Spline Area chart in AnyChart.

To learn more about vertical charts in general and how to customize them, see Vertical Charts (Overview). You can also read the Range Spline Area Chart article to learn about other available settings.

Quick Start

To build a Vertical Range Spline Area chart, use the anychart.verticalArea() or anychart.vertical() chart constructor. Then call the rangeSplineArea() method to create a Range Spline Area series.

// create a chart
chart = anychart.verticalArea();

// create a range spline area series and set the data
var series = chart.rangeSplineArea(data);
